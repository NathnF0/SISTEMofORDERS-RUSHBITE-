## 🚀 Sobre o Projeto
O **RushBite** é um ecossistema de delivery completo via terminal, focado em resolver dores reais de negócio. O projeto simula a interação entre clientes e lojistas, com lógica de cupons, taxas de entrega e inteligência financeira.

### 🎥 O que há de novo na v0.2.5?
Nesta versão, saímos do básico e entramos na **Gestão Profissional**:
* **🏦 Dashboard Financeiro:** Controle de faturamento líquido e ticket médio.
* **🎫 Cupons Híbridos:** Validação de cupons da plataforma (Sazonais: Páscoa/Natal) e cupons exclusivos criados pelas lojas.
* **📂 Menus Verticais:** Interface limpa, organizada por seções e cores.
* **💬 Chat de Status:** Histórico de mensagens entre loja e cliente integrado ao pedido.
